# برنامج أجور الراجحي Premium v4

برنامج Desktop لإدارة العمال والرواتب والبدلات وتوليد ملف أجور TXT / CSV لبنك الراجحي.

## يغطي المتطلبات
- يعمل كبرنامج Desktop على Windows.
- استيراد ملف Excel الخاص بالعمال.
- حفظ بيانات العمال داخل البرنامج.
- تعديل راتب كل عامل على حدة.
- تعديل البدلات والخصومات.
- تعديل جماعي للرواتب/البدلات عند الحاجة.
- توليد ملف TXT للأجور.
- توليد CSV احتياطي.
- حفظ المسيرات السابقة.
- تشغيل 3 أجهزة على نفس الشبكة عبر قاعدة بيانات مشتركة.
- يدعم أعداد كبيرة من العمال مع بحث سريع.

## طريقة تشغيل أكثر من جهاز
1. على الجهاز الرئيسي أنشئ فولدر مشترك Shared.
2. من إعدادات البرنامج اختر قاعدة بيانات مشتركة داخل هذا الفولدر.
3. من باقي الأجهزة اختر نفس ملف قاعدة البيانات.

## استخراج EXE من GitHub
ارفع محتويات المشروع، ثم Actions > Build Windows EXE > Run workflow.
